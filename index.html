About GNU MAGIC

GNU MAGIC is a fast method to calculate solar irradiance (beam and diffuse) for large areas. It uses look up tables containing results from libRadtran, a radiative transfer model, to efficiently and accurately determine solar irradiance at the earth's surface. It also contains an interface for satellite based cloud information.

GNU MAGIC has been used by CM SAF to produce essential climate variable datasets from Meteosat imagery dating back to 1983. It was also used to produce the European and African solar atlas, PVGIS. It is currently being expanded to allow the use of further satellites.

How does it work?

GNU MAGIC is well documented, both in peer reviewed publications and in technical documents. Visit the wiki for information about how to use GNU MAGIC, what scientific principles it uses and how well it validates against ground-based measurements.
